About MoreGDT


This mod adds all kinds of new platforms, events, topics, storylines and more in order to make Game Dev Tycoon more interesting and create a better player environment.


New platforms

In MoreGDT there are many new platforms! Extended Grapple platforms include newer grPhones, newer grPads and introduces the Grapple Grapintosh line of product from the 128k up to the Imtel iGrap! A new Nexis line of products starting with the Nexis S and leading right up to the Nexis 5X/6P, followed by the Voogle Pikzel. The vNidia Armour Portable and Armour TV are also new platforms you can develop games for in MoreGDT!

New topics and events

New topics allow you to create a plethora of amazing games that you otherwise wouldn't be able to in Game Dev Tycoon. There are more topics added to MoreGDT with each update as well as events that make the game more interesting.

New storylines

MoreGDT adds something more than new products, topics and events... It adds storylines and other things that you will see happening as you progress through Game Dev Tycoon an example would be different versions of Vindows being released every now and then showing how the PC platform's technology improves over time.
